# SLD Conventions and Symbology Reference

This document describes the symbols and conventions used in single-line
diagrams generated by the electrical-distribution-skill.

## Voltage Level Colors

| Voltage Range | Color | Usage |
|---------------|-------|-------|
| 11kV - 33kV | Red (#FF0000) | Medium Voltage |
| 3.3kV - 6.6kV | Orange (#FF6600) | Medium Voltage |
| 480V - 690V | Blue (#0066FF) | Low Voltage (Industrial) |
| 380V - 415V | Royal Blue (#4169E1) | Low Voltage (Standard) |
| 120V - 240V | Green (#228B22) | Low Voltage (Distribution) |

## View Modes

### System View (Collapsed)

Shows buses as single blocks with aggregated load information:
- Bus name and voltage
- Total connected kW
- Number of feeders
- No individual equipment shown

Best for:
- System overview
- Utility coordination drawings
- Tier 1 (preliminary) designs

### Detailed View (Expanded)

Shows individual equipment within each bus:
- Individual motor/load details
- Feeder types (DOL, VFD, etc.)
- Power ratings
- Equipment tags

Best for:
- Engineering review
- Panel layout reference
- Tier 2/3 (detailed) designs

## Label Conventions

### Equipment Labels

```
{Tag}
{Description}
{Rating} kW
{Feeder Type}
```

Example:
```
200-B-01A
Aeration Blower #1
90 kW
VFD
```

### Transformer Labels

```
{Tag}
{Rating} kVA
{Primary}/{Secondary}
```

Example:
```
XFMR-01
1000 kVA
11kV/400V
```

### Bus Labels

```
{Name}
{Voltage} {Rating}A
```

Example:
```
MCC-200 Biological
400V 630A
```

## IEC vs NEMA Conventions

This skill supports both IEC and NEMA conventions based on the
`code_basis` setting in the electrical basis:

| Aspect | IEC | NEMA |
|--------|-----|------|
| Motor ratings | kW | HP |
| Voltage | 400V, 415V | 480V, 575V |
| Frequency | 50 Hz | 60 Hz |
| Protection | IEC 60947 | UL/NEC |
| Motor FLC tables | IEC 60034 | NEC 430.250 |
